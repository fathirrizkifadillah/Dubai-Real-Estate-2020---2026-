## Dataset Description

This dataset contains monthly real estate indicators across multiple Dubai communities. It includes property prices, rental market information, listing volumes, and macroeconomic indicators that can be used to analyze market trends and forecast future property values.

| Column                             | Description                                                                     |
| ---------------------------------- | ------------------------------------------------------------------------------- |
| `year_month`                       | Observation month in YYYY-MM format.                                            |
| `community`                        | Name of the property community or neighborhood.                                 |
| `zone`                             | Broader geographic area or zone where the community is located.                 |
| `is_freehold`                      | Indicates whether the area allows full property ownership by foreign investors. |
| `secondary_price_per_sqft_usd`     | Average secondary-market property price per square foot (USD).                  |
| `secondary_price_per_m2_usd`       | Average secondary-market property price per square meter (USD).                 |
| `offplan_price_per_sqft_usd`       | Average off-plan property price per square foot (USD).                          |
| `rental_price_per_sqft_annual_usd` | Average annual rental price per square foot (USD).                              |
| `n_listings_secondary`             | Number of active secondary-sale property listings.                              |
| `n_listings_offplan`               | Number of active off-plan property listings.                                    |
| `n_listings_rental`                | Number of active rental property listings.                                      |
| `cbuae_base_rate_pct`              | Central Bank of the UAE (CBUAE) base interest rate (%).                         |
| `avg_mortgage_rate_pct`            | Average mortgage interest rate (%).                                             |

### Potential Analysis Areas

* Property price trends over time
* Community-level market comparison
* Rental market analysis
* Supply and demand dynamics
* Impact of interest rates on property prices
* Freehold vs non-freehold market comparison
* Property price forecasting
* Investment opportunity identification
