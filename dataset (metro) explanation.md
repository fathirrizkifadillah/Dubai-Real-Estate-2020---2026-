## Dataset Description

This dataset contains information about Dubai Metro stations, including their locations, metro lines, opening years, and distances from Burj Khalifa. It can be used to analyze transportation accessibility and its potential relationship with real estate prices.

| Column               | Description                                                             |
| -------------------- | ----------------------------------------------------------------------- |
| `station_name`       | Name of the Dubai Metro station.                                        |
| `line`               | Metro line to which the station belongs (e.g., Red Line or Green Line). |
| `lat`                | Latitude coordinate of the station.                                     |
| `lon`                | Longitude coordinate of the station.                                    |
| `year_opened`        | Year when the station began operation.                                  |
| `to_burj_khalifa_km` | Distance from the station to Burj Khalifa in kilometers.                |

### Business Relevance

This dataset can be used to investigate the impact of transportation infrastructure on real estate values and urban development.

Potential analysis includes:

* Distribution of metro stations across Dubai
* Comparison between Red Line and Green Line coverage
* Accessibility analysis based on distance to central business areas
* Relationship between metro proximity and property prices
* Identification of communities with strong transportation connectivity
* Urban growth patterns around metro corridors
