## Dataset Description

This dataset contains information about off-plan real estate projects in Dubai. Off-plan properties are units that are sold before construction is completed, making them a key segment for investors seeking capital appreciation and flexible payment plans.

The dataset includes project details, developer information, property characteristics, location data, accessibility metrics, pricing information, and financing conditions.

| Column                     | Description                                                            |
| -------------------------- | ---------------------------------------------------------------------- |
| `id`                       | Unique identifier for each property listing.                           |
| `date_listed`              | Date when the property was listed.                                     |
| `project_id`               | Unique identifier for the development project.                         |
| `project_name`             | Name of the real estate development project.                           |
| `developer`                | Name of the project developer.                                         |
| `developer_tier`           | Developer classification based on market reputation or scale.          |
| `community`                | Community or neighborhood where the project is located.                |
| `zone`                     | Broader geographic area or district.                                   |
| `lat`                      | Latitude coordinate of the property location.                          |
| `lon`                      | Longitude coordinate of the property location.                         |
| `property_category`        | General property category (e.g., apartment, villa).                    |
| `property_type`            | Specific property type and bedroom configuration.                      |
| `bedrooms`                 | Number of bedrooms.                                                    |
| `area_sqft`                | Property size in square feet.                                          |
| `area_m2`                  | Property size in square meters.                                        |
| `view`                     | Primary view offered by the property (e.g., sea, city, pool).          |
| `launch_year`              | Year when the project was launched.                                    |
| `handover_year`            | Expected project completion and delivery year.                         |
| `total_project_units`      | Total number of units within the development.                          |
| `payment_plan`             | Payment structure offered by the developer.                            |
| `metro_station`            | Nearest Dubai Metro station.                                           |
| `metro_line`               | Metro line associated with the nearest station.                        |
| `metro_distance_min`       | Estimated travel time or distance metric to the nearest metro station. |
| `to_burj_khalifa_km`       | Distance to Burj Khalifa in kilometers.                                |
| `price_usd`                | Total property price in USD.                                           |
| `price_per_sqft_usd`       | Price per square foot in USD.                                          |
| `price_per_m2_usd`         | Price per square meter in USD.                                         |
| `mortgage_rate_at_listing` | Mortgage interest rate at the time the property was listed.            |

### Business Relevance

This dataset enables analysis of Dubai's off-plan market from both investment and development perspectives.

Potential analysis includes:

* Most expensive and affordable projects
* Developer performance comparison
* Property type and bedroom pricing analysis
* Impact of location on property values
* Influence of metro accessibility on prices
* Relationship between project size and pricing
* Off-plan investment opportunities
* Payment plan trends across developers
* Forecasting future project values
* Market segmentation by developer tier
